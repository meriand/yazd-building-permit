Yazd Building Permit Management System
A Scrumban-driven platform for handling building permit applications within Yazd Municipality.

Project Overview
This system oversees the complete lifecycle of building permit applications, from submission through review and final approval, ensuring efficient tracking and management of requests.

Tech Stack

Backend: Node.js with Express

Database: MongoDB (using Mongoose)

Project Management: Scrumban methodology

Version Control: GitHub with feature-branch workflow

Features

Submit new building permit requests online

Track the status of requests in real-time

Review and approve permits efficiently

Generate reports and maintain historical records
